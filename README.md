## 安装步骤
参考 <https://github.com/yaocccc/omz/tree/master>
1. 依赖安装
    ```bash
    # 安装 fzf
    https://github.com/junegunn/fzf
    # 安装 lua
    https://github.com/lua/lua
    # 安装 trashy
    https://github.com/oberblastmeister/trashy
    # 安装 zsh
    ```
2. 克隆仓库
    ```bash
    # 克隆本仓库
    git clone https://github.com/wuqin0202/dotfiles.git
    cd dotfiles
    ```
3. 开始安装
    ```bash
    chsh -s $(which zsh)
    ./install.sh
    ```
